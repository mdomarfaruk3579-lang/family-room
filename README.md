# 🌳 Family Room - Interactive Family Tree

একটি আধুনিক এবং ডাইনামিক ফ্যামিলি ট্রি অ্যাপ, যা রিয়্যাক্ট (React) এবং টেলউইন্ড সিএসএস (Tailwind CSS) দিয়ে তৈরি। এটি খুব দ্রুত লোড হয় এবং মোবাইল ফ্রেন্ডলি।

## ✨ মূল ফিচারসমূহ:
- **Click-to-Expand:** প্রতিটি জেনারেশন কার্ডে ক্লিক করলে পরবর্তী প্রজন্ম স্মুথভাবে দেখা যায়।
- **Smooth Animations:** ফ্রেমার মোশন (Framer Motion) ব্যবহার করে মাখনের মতো মসৃণ ট্রানজিশন।
- **Zero-Scrollbar UI:** ব্রাউজারের বিরক্তিকর স্ক্রলবার ছাড়াই একটি ক্লিন এবং আধুনিক ইন্টারফেস।
- **Mobile Optimized:** যেকোনো ব্রাউজার এবং মোবাইল ডিভাইসে সুপার-ফাস্ট পারফরম্যান্স।

## 🛠 প্রযুক্তি (Tech Stack):
- **Frontend:** React.js, Tailwind CSS
- **Animations:** Framer Motion
- **Deployment:** Vercel

## 🚀 প্রোজেক্টটি কীভাবে চালাবেন:
১. জিপ ফাইলটি আনজিপ করুন।
২. টার্মিনালে `npm install` কমান্ড দিয়ে প্রয়োজনীয় প্যাকেজ ইন্সটল করুন।
৩. `npm run dev` লিখে আপনার লোকাল সার্ভারে অ্যাপটি ওপেন করুন।
